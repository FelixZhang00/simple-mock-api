#sample-mock-api

Use [json-server](https://github.com/typicode/json-server) make this sample mock api server.





### Usage
	
1. checkout this project,and cd
2. type 
```
npm install
```
in terminal to install dependences

3. run
```
node app.js
```
4. get result via your configed cgi
